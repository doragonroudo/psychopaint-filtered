## This repo consists of 2 parts
- `model-api` : API for AI's prediction **No exported models in repo** (using `Tensorflow` and `Flask`)
- `psycho-app` : Web App (using `React Native Expo CLI`)

## How to run Mobile App
    cd psycho-app
    npm install
    npm start

## How to run API
    cd model-api
    python server.py
